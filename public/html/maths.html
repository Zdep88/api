Homothety

GET

Description

Calculate the homothety of a point with respect to a center and transformation parameters.

Endpoint

https://api.zdep.fr/maths/homothety

Input Method

Query Parameters

Parameters

pointX
X coordinate of the point to transform (default: 0)
pointY
Y coordinate of the point to transform (default: 0)
centerX
X coordinate of the center of homothety (default: 0)
centerY
Y coordinate of the center of homothety (default: 0)
rotate
Rotation angle in degrees (default: 0) - Use minus sign for counter-clockwise rotation or if y-axis has positive values downwards
zoom
Zoom factor (default: 1)

Request Example

https://api.zdep.fr/maths/homothety?pointX=3&pointY=3¢erX=2¢erY=3&rotate=90&zoom=2

Response Example

{
  "pointX": 2,
  "pointY": 5
}
